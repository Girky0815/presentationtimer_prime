# プレゼンタイマー Prime - プロジェクト構成 (lib/)

リファクタリング後の `lib` ディレクトリ以下のファイル構成と各ファイルの役割です。

## ディレクトリ構成

### 📂 lib/
アプリケーションの主要なソースコードが格納されています。

- **`main.dart`**: アプリケーションのエントリーポイント。 `main()` 関数のみを含みます。
- **`app.dart`**: アプリケーションのルートウィジェット (`PresentationTimerApp`)。テーマ設定、ローカリゼーション、プロバイダーの初期化を行います。

### 📂 lib/models/
データモデルクラスをまとめています。

- **`bell_config.dart`**: ベル（時間経過通知）の設定情報を保持するデータクラス。

### 📂 lib/screens/
アプリの各画面（ページ）となるウィジェットをまとめています。

- **`timer_screen.dart`**: メイン画面。タイマーの表示、操作、ベルの通知を行います。
- **`settings_panel.dart`**: 設定画面。外観やタイマーのデフォルト値の設定を行います。
- **`color_scheme_preview_screen.dart`**: カラーパレット確認用のデバッグ画面。

### 📂 lib/services/
データの永続化などのサービスロジックをまとめています。

- **`preferences_service.dart`**: `shared_preferences` を使用して設定やベル情報を保存・読み込みする機能を提供します。

### 📂 lib/state/
アプリケーションの状態管理（State Management）をまとめています。

- **`timer_state.dart`**: `ChangeNotifier` を継承した状態管理クラス。タイマーのロジック、設定値、テーマモードなどを一元管理します。

### 📂 lib/theme/
アプリのデザインテーマに関する定義をまとめています。

- **`color_schemes.dart`**: ライトモード・ダークモードのカラーパレット定義。
- **`typography.dart`**: テキストスタイル（フォント、サイズ、太さなど）の定義と生成ロジック。

### 📂 lib/utils/
汎用的なユーティリティ関数をまとめています。

- **`time_formatter.dart`**: 時間（秒数）を「分:秒」形式の文字列に変換する関数などを提供します。

### 📂 lib/widgets/
複数の画面で使用される、あるいは特定の機能を持った再利用可能なウィジェットコンポーネントをまとめています。

- **`animated_mode_switcher.dart`**: タイマー/ストップウォッチのモードを切り替えるためのアニメーション付きスイッチ。
- **`bell_chip.dart`**: 設定されたベルを表示するチップ型のウィジェット。
- **`bell_edit_dialog.dart`**: ベルの時間や回数を編集するためのダイアログ。
- **`expressive_section.dart`**: 設定画面などで項目をグループ化して表示するためのコンテナウィジェット。
- **`time_input_box.dart`**: 時間入力（数値入力）を行うためのテキストフィールドウィジェット。
